# 单测项趋势转折（速率变化）预报效能分析

对单个测项观测序列做趋势转折异常识别与预报效能评估：读取观测数据与地震目录，经预处理、归一化与滑动窗口矢量转角极值提取，在「阈值×预报期」网格上计算 R 值，输出 Rmax、R0、报准/漏报统计及图件。

## 1. 程序与依赖

- **主程序**：`trend_anomaly_single.py`（单测项完整流程）
- **依赖安装**（建议 Python 3.9+）：

```bash
pip install -r requirements.txt
```

- **可选**：若需在空间分布图中使用 **cartopy** 底图，在程序中设置 `USE_CARTOPY_MAP = True` 并安装：`pip install cartopy`。

## 2. 输入文件


| 文件           | 说明                                                                                     |
| ------------ | -------------------------------------------------------------------------------------- |
| **观测数据**     | 文本，两列：`时间标签`、`观测值`。时间可为 `yyyymmdd`（日值）或 `yyyymmddhh`（小时值，程序内转日均）；或多列（日期 + 24 小时值，取日均）。 |
| **多边形规则**    | 文本，首行为标题（可含中文），其后两列：`经度`、`纬度`，每行一个顶点，首尾闭合。                                             |
| **地震目录**     | EQT 定长格式，每行包含年月日时分秒、经纬度、震级、深度等。                                                        |
| **干扰信息**（可选） | 文本文件。首行为标题，前两列：干扰开始时间、结束时间（时间码格式同观测数据）。不存在或为空时按无干扰处理。                                  |


观测数据、多边形、地震目录**内容为空**时程序会报错并退出；干扰文件可为空或不存在。无震例时仅屏幕提示结束，不写结果与图件。

## 3. 配置与运行

在 `trend_anomaly_single.py` 末尾 `if __name__ == "__main__":` 中修改以下参数后运行：

```python
DATA_FILE = Path(r"你的观测数据路径.txt")
POLYGON_FILE = Path(r"你的多边形规则路径.txt")
EQ_CATALOG_FILE = Path(r"你的地震目录路径.eqt")
INTERFERENCE_FILE = Path(r"干扰信息路径.txt")   # 无则填 None
STATION_LON = 台站经度
STATION_LAT = 台站纬度
MAG_MIN = 预测震级下限
MAG_MAX = 预测震级上限
WINDOWS_LENGTH = 180   # 滑动窗口长度（天）
WINDOWS_STEP = 1       # 滑动步长（天）
DAY_S, DAY_N, DAY_E = 1, 1, 360   # 预报有效期下限、步长、上限（天）
THRES_S, THRES_N, THRES_E = 50, 1, 180   # 阈值扫描范围（度）
USE_CARTOPY_MAP = False   # True 时需安装 cartopy
```

运行：

```bash
python trend_anomaly_single.py
```

## 4. 流程概览

1. 读取观测数据；缺数插值（线性或 PCHIP）、台阶修正；若为小时值则转日均值。
2. 归一化后滑动窗口左右线性拟合求矢量转角，取峰得到角度极值序列。
3. 按阈值筛选异常时间点，剔除落在干扰时段内的点。
4. 输出处理序列文件：`<数据文件名>_processed.txt`（Time, Fitted_Angle, Is_Anomaly）。
5. 按多边形、预测震级范围、观测时间范围等从地震目录中筛选震例；有震例时继续，无震例则提示结束。
6. 在 (阈值×预报期) 网格上计算 R 矩阵，取 Rmax 及对应阈值、预报期；计算 R0 与报准/漏报。
7. 写结果文件 `<数据文件名>_result.txt`，并生成图件 `<数据文件名>_result.png`。

## 5. 输出文件

- `**<数据文件名>_result.txt**`：趋势转折异常与预报效能结果（Rmax、R0、最优阈值与预报期、异常起止、报准/漏报及震例信息、震中距等）。
- `**<数据文件名>_processed.txt**`：三列，分别为 Time（YYYYMMDD）、Fitted_Angle（拟合角度极值，保留 3 位小数）、Is_Anomaly（是否异常，1/0）。
- `<数据文件名>_result.png`：四幅子图（预处理曲线与异常点及拟合范围框、空间分布、转角极值与阈值/预报窗/震例、R–TT 曲面）。

所有输出文件与观测数据文件同目录。
