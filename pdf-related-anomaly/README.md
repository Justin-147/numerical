# 基于概率密度拟合的异常分析（pdf-related anomaly）

本程序主要功能：对分钟采样观测时序做小波去趋势，在滑动窗口内对标准化序列的概率分布拟合参数 λ，得到时序 λ² 及拟合误差，用于异常识别。

## 1. 本 Python 程序

- **主程序**：`pdf_anomaly.py`
- **程序入口**：在文件末尾 `if __name__ == "__main__":` 中配置 `DATA_FILE` 等参数后运行：
  ```bash
  python pdf_anomaly.py
  ```

### 1.1 缺测与窗长说明

- **缺测**：99999、999999 等视为缺测，**直接剔除对应行，不做插值**。
- **窗长含义**：滑动窗口按**采样点数**（如 `30×24×60` 个点）定义，不是按“30 个自然日”定义。剔除缺测后，序列在时间上不再等间隔，因此**同一窗长（点数）对应的实际时间跨度可能不相等**，各窗覆盖的日历范围会因缺测位置而不同。

### 1.2 功能概览

1. **数据读取**：两列文本（时间码 + 观测值），自动剔除 99999、999999 等缺测。
2. **小波去趋势**：取指定层小波细节作为“高频”序列。
3. **滑动窗口**：按窗长、步长滑动，窗内标准化后做经验 PDF（-4~4，0.1 步）→ 积分型理论 PDF 拟合得到 λ，输出 λ² 与误差。
4. **输出**：与数据文件同目录（或指定 `output_dir`）：
  - `{stem}_{level}_{wavelet}_detrend.txt`：去趋势序列（时间\t数值），如 `yasw_4_db5_detrend.txt`
  - `{stem}_{level}_{wavelet}_lmd.txt`：时间、λ²、拟合误差，如 `yasw_4_db5_lmd.txt`
  - `{stem}_{level}_{wavelet}.png`：原始值、去趋势、λ² 三幅子图

### 1.3 可调参数（在 `__main__` 中）


| 参数               | 说明                   | 默认   |
| ---------------- | -------------------- | ---- |
| `DATA_FILE`      | 输入数据文件路径             | 示例路径 |
| `OUTPUT_DIR`     | 输出目录（None 则与数据文件同目录） | None |
| `WAVELET`        | 小波名                  | db5  |
| `LEVEL`          | 小波分解层数               | 4    |
| `WINDOW_MINUTES` | 窗长（分钟），None 则 30 天   | None |
| `STEP_MINUTES`   | 步长（分钟），None 则 1 天    | None |


### 1.4 输入数据格式

- 文本文件，两列：第 1 列时间码，第 2 列分钟采样观测值。
- 时间码支持：**12 位** `yyyymmddHHMM`。
- 缺测用 99999 或 999999 表示，程序会自动剔除。

## 2. 依赖安装

```bash
pip install -r requirements.txt
```

建议 Python 3.8+。

## 4. 输出文件说明

- `***_detrend.txt**`（如 `yasw_4_db5_detrend.txt`）：去趋势后序列（时间、数值）。
- `***_lmd.txt**`（如 `yasw_4_db5_lmd.txt`）：每行「时间、λ²、拟合误差」，可用于后续异常判定或成图。
- `***.png**`（如 `yasw_4_db5.png`）：上=原始值，中=去趋势，下=λ² 时序。
